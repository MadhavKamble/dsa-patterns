## 📖 How to Study a Pattern

1. **Learn** — read the pattern explanation and ASCII visual (30 min)
2. **Internalize** — memorize the recognition trigger and core idea (15 min)
3. **Code** — study the C++ template in `Templates/`, then code problems (45–90 min)
4. **Drill** — do pattern recognition drills without hints (15 min)
5. **Log** — update [../PROGRESS.md](../PROGRESS.md) with confidence rating
