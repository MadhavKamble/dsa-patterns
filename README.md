## ⚡ Pattern Quick-Reference

| If you see... | Pattern |
|---------------|---------|
| "subarray", "substring", "window" | Sliding Window |
| Sorted array, two elements summing to target | Two Pointers (converging) |
| Linked list cycle, find middle | Fast & Slow Pointers |
| Level-by-level, right side view | BFS Level Order |
| Diameter, max depth, path sum | DFS Postorder |
| Number of islands, connected components | DFS/BFS |
| Course schedule, prerequisites | Topological Sort |
| "minimum/maximum satisfying condition" | Binary Search on Answer |
| Top K, K closest, K largest | Heap |
| All subsets, combinations, permutations | Backtracking |
| Merge intervals, meeting rooms | Greedy Intervals |
| Next greater element, temperatures | Monotonic Stack |
| Edit distance, longest common subsequence | DP (String) |
| Coin change, ways to make sum | Unbounded Knapsack DP |
| Prefix of words, autocomplete | Trie |
